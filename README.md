# Myshell
help              : các lệnh trong myshell.  
fore 'nameprogram': chạy chương trình dạng foreground mode.  
back 'nameprogram': chạy chương trình dạng background mode.  
list              : danh sách tiến trình.  
kill 'processid'  : dừng, xóa tiến trình.  
stop 'processid'  : tạm dừng tiến trình đơn luồng.  
resume 'processid': tiếp tục tiến trình đơn luồng.  
exit              : kết thúc chương trình myshell.
